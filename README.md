# mission-tree-maker
A mission tree maker, mainly meant for Astroneer


# Canvas Controls
Ctrl+P:
Select an ID, then use arrow keys to move

Left Click:
When over a mission, open mission editor, otherwise create a new mission

Movement:
WASD, or use the mission mover with ARROW-KEYS

E:
Checks a mission, marking it complete. Press again to turn off.

Right Click:
Delete Mission

Save & Load:
Ctrl+S to save, Ctrl+L to load.

# Editor Controls

Tab:
Switch the Selected Field

Mission Type:
Type N/S to switch between Normal and Special


# YOU NEED PYTHON INSTALLED FOR THIS SCRIPT TO WORK
Logic Type:
Press T to toggle AND/OR

Copy & Paste:
Ctrl+C copies to the built-in clipboard, Ctrl+V pastes from it, replacing the current field with the clipboard data.
